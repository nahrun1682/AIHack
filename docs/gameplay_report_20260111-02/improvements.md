## 新たな課題と提案

### 1. ゲーム攻略の多様性
*   **現状**: Stage 2の敵が単純な権威付け（General）では落ちないことが確認された（適正バランス）。
*   **課題**: ユーザーが次にどのような手を打てばいいかの「気づき」が得にくい。
*   **提案**:
    *   **スカウティング機能**: 敵のSystem Promptの一部（性格など）を「偵察」するコマンドやアップグレード。
    *   **論理攻撃ボーナス**: 「緊急事態」や「権威」だけでなく、「論理的矛盾」を突くようなプロンプトに対して味方AIが補正を掛ける仕組み。

### 2. さらなるUI改善
*   **課題**: 右カラムのログが長くなると、最新の会話が下の方に行ってしまう（スクロールが必要になる場合がある）。
*   **提案**:
    *   チャットログを「新しい順」に表示するオプション、または自動スクロール（Streamlitの仕様上難しい場合があるが要検討）。
    *   ストリーミング中の自動スクロール追従。

### 3. コンテンツ
*   **提案**: Stage 3以降のギミックとして、単語禁止だけでなく「文字数制限が極端に短い」「特定の言語しか話さない」などのバリエーション追加。
